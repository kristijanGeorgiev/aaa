Build Form with the following interface:
a) Username TextFormField
b) Phone Number TextFormField
c) Age TextFormField
d) Submit Button
Tasks:
a) On Submit Button press validate Username, Age and Phone user inputs
b) Validations: 
All fields are mandatory, 
Username: 4-10 characters, must start with a letter and contain no numbers 
Phone Number: must start with 389 and be exactly 11 digits 
Age: must be between 18-99 and must not accept leading zeros
c)  if all inputs pass validation, display an AlertDialog with the message; "Profile saved for [Username]".
d) if any field is invalid, display a SnackBar with the message: "Check input fields for errors!"
